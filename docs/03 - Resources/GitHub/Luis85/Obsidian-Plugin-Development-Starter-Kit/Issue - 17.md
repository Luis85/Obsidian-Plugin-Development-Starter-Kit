A README.md in a Project is not "just a welcome file" - its the front door of the project. 

It should function as a Project Brief + User Guide + Entry Point, helping different stakeholders (devs, designers, PMs, leadership, auditors) quickly understand what this project is, why it exists, and how to engage with it.
