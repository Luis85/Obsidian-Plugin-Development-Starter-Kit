# Collaboration & Contribution
## How can I help?

- Search for Issues marked with help needed
- Review the Issue and offer Help or ask for Clarification
- Review active Pull Requests and give Feedback or ask for Clarification


## Collaboration

Before opening a new Pull Request, look into the Issues if there is already an Item fitting the Problemspace you want to tackle.
If there is, please feel free to present your Idea in the coresponding Issues Comment Section and lets help you bringing your changes to main.
If not, Branch out and Create a Draft Pull Request to open the Discussion.

## Contribution

We us the GitHub Pull Requests Workflow.

- Git Clone the repository
- Create new Branch
- Create a new Pull Request in Draft, pointing to that Branch
- Reference the Issues you want to improve
- Describe your Contribution in the Problem->Insight->Solution Format
- Ask for Review on a regular basis
